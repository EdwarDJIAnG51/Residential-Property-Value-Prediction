# Residential-Property-Value-Prediction
Multi-modal stacking ensemble for residential property value prediction (score 0–100). Combines XGBoost, LightGBM, and CatBoost on tabular features with OpenAI CLIP (ViT-B/32) image embeddings and TF-IDF text features, stacked via Ridge regression. Achieved validation RMSE of 3.26. Built for the WUSTL T81-558 Kaggle competition (Fall 2025).
